Lesson 1

Notes

localStorage is a way to store information on a page that a user has input, even when they refresh the page. localStorage can also store the information until they come back later.

'var' is a variable that you want to lable a group of objects as. It's can be compared to a box filled with similar/the same item that is labeled so that you can easily find that item later.

When you declare a variable and tell it to get a certain element (getElementById/querySelector), you are asking it to grab that specific element, take what's inside of it, and store that information into the variable.

Code Examples

if (localStorage.clickcount) { localStorage.clickcount = Number(localStorage.clickcount) + 1; } else { localStorage.clickcount = 1; } document.getElementById("result").innerHTML = "You have clicked the button " + localStorage.clickcount + " time(s).";
